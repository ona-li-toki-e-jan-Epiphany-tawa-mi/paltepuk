## Compiler Macros

The following predefined macros can be used in C/C++ to check if a program is
being compiled for DOS:

- MSDOS
- \_\_MSDOS\_\_
- \_MSDOS
- \_\_DOS\_\_

Sourced from
[https://github.com/cpredef/predef *(May 19, 2025)*](https://github.com/cpredef/predef).
